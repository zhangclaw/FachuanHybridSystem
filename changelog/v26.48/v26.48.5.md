## [26.48.5] - 2026-05-11

### 前端

#### 新功能

- **批量分析下载优化**：下载按钮从图标改为"下载CSV"/"下载ZIP"文字按钮，点击弹出确认框支持选择"仅相关案例"或"全部案例"下载
- **合同立案标签页精简**：移除建档信息卡片，保留律所 OA 和关联案件

#### 修复

- **批量分析取消后汇总缺失**：取消任务时仍为已完成的案例生成汇总报告，前端 cancelled 状态也显示汇总消息和下载按钮

### 后端

#### 新功能

- **CSV 详细分析列**：批量分析汇总 CSV 新增"详细分析"列，包含完整的分析正文内容
- **下载按相关性筛选**：下载 API 支持 `?relevant_only=true` 参数，实时过滤生成仅包含相关案例的 CSV/ZIP
- **Query parameter token 认证**：`JWTOrSessionAuth` 支持 `?token=xxx` 查询参数认证，修复 `window.open` 下载场景的 401 错误

#### 修复

- **LLM 后端可用性检查**：`is_available()` 现在正确尊重 `config.enabled` 标志，未启用的后端不再返回可用
- **批量分析降级逻辑**：不再错误降级到未配置的 LLM 后端
