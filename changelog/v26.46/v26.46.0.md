## [26.46.0] - 2026-05-08

### 前端

#### 新功能

- **工作台 AI 交互体验优化**：工具调用结果支持结构化渲染（案件、客户、公司、合同、列表、提醒），法律文本自动格式化，支持上下文附件
- **SSE 断线自动重连**：工作台流式对话断线后自动重试（最多 3 次），界面显示重连状态提示
- **404 页面**：新增 NotFoundPage，未匹配路由统一展示 404 提示并支持返回首页
- **联系人搜索集成聚合搜索**：移除独立联系人搜索页面，联系人结果直接展示在全局聚合搜索中
- **案件工作人员联系方式**：案件详情页新增 CaseContactSection 组件，支持增删改查案件关联的工作人员联系方式
- **仪表盘统计模块**：新增统计卡片（StatsCards）、趋势图（TrendChart）、日历卡片（CalendarCard）、案件分布圆环图（CaseDistributionChart）
- **DOC 转 DOCX 工作台**：管理后台新增文档转换工作台页面，支持文件上传转换
- **工具调用结果组件**：新增 CaseResult、ClientResult、CompanyResult、ContractResult、ListResult、ReminderResult 六个工具结果渲染组件

#### 修复

- **切换会话自动滚到底部**：切换工作台会话后自动滚动到最新消息
- **消息显示时间戳**：消息气泡增加时间戳显示
- **律师头像显示**：修复律师详情页和导航栏不显示头像的问题
- **工作台欢迎内容居中**：修复欢迎内容未垂直居中的问题
- **multipart 文件上传 422**：修复文件上传接口返回 422 错误
- **圆环图 tooltip 叠加**：修复案件类型分布圆环图 tooltip 文字与图表叠加
- **趋势图轴标签间距**：优化趋势图 X/Y 轴标签间距避免重叠

#### 优化

- **移除仪表盘冗余模块**：移除即将到期提醒和快捷操作模块，简化仪表盘布局

### 后端

#### 新功能

- **案件工作人员联系方式**：新增 contacts 应用，CaseContact 模型 + Admin + API + Service 全套实现，支持自动补全搜索主管机关
- **DOC 转 DOCX 工作台**：管理后台文档转换工作台页面，支持文件上传和转换状态追踪
- **仪表盘统计服务**：dashboard_service 提供案件、客户、合同等维度的统计数据和趋势分析
- **批量分析详情 ZIP 下载**：每个案例生成独立 .md 分析报告，打包为 ZIP 下载
- **全局搜索扩展**：search_api 增加联系人搜索结果
- **DOC 转换清理命令**：cleanup_stale_doc_converter_jobs 管理命令，清理过期转换任务

#### 修复

- **CaseContact admin clean()**：从 cleaned_data 获取 case 实例，避免表单验证错误
- **CaseContact admin 菜单隐藏**：CaseContact 从 Django admin 侧边栏隐藏，仅通过案件详情页内联管理
- **dashboard API Pydantic 验证**：修复 Django lazy translation string 导致 Pydantic 验证失败
- **mypy 类型错误**：修复 values() 查询结果的类型注解问题
- **批量分析 ZIP 按需生成**：修复旧任务（ZIP 功能上线前创建）下载详情 ZIP 返回 404 的问题，改为按需生成
- **multipart 文件上传 422**：修复文件上传接口参数解析错误
- **contact-search 路由清理**：移除已废弃的联系人搜索路由，使其正确返回 404
