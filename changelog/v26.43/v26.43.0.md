## [26.43.0] - 2026-05-05

### 前端

#### 新功能

- **Cmd+K 全局搜索**：新增 `CommandPalette` 组件，支持通过快捷键唤起全局搜索，可检索案件、当事人、合同等业务数据
- **案件管理全面对齐后端**：
  - 新增案件材料管理（CaseMaterialSection），支持材料列表、上传、绑定与候选材料推荐
  - 新增案件文件夹管理（CaseFolderSection），支持文件夹创建、绑定与解绑
  - 新增案件模板管理（CaseTemplateSection），支持模板绑定与管理
  - 新增案件访问授权（CaseAccessGrantSection），支持按律师授予访问权限
  - 案件详情页（CaseDetail）全面改版，集成上述所有新 Section
  - 案件日志（CaseLogSection）改进
- **合同管理增强**：
  - 合同详情页全面改版，新增归档 Tab、文书 Tab、费用 Tab、立案 Tab
  - 合同列表搜索和筛选参数修复（已正确传递到后端）
- **设置页面体系**：
  - 新增设置概览页、律所设置、律师设置、团队设置
  - 新增服务配置页（ServiceConfig），支持外部服务集成配置
  - 新增操作日志页（LogsPage）
  - 新增任务队列管理页（TaskQueuePage），支持查看和管理后台任务
- **模板管理模块**：新增模板列表、新建/编辑表单、模板详情页，完整的 CRUD 功能
- **工具模块大幅扩展**：
  - 法院短信工具：完整的提交、查看列表、详情查看、重试功能，对接后端 API
  - LPR 利率计算器：对接后端 API，支持利率查询与计算
  - 快递查询工具（CourierTrackingTool）：对接后端 EMS/顺丰查询 API
  - 要素转换工具（ElementConvertTool）
- **消息来源管理**：新增消息来源列表页面，对接后端 MessageSource API
- **登录注册完善**：
  - 新增忘记密码页面（ForgotPasswordPage）
  - 新增重置密码页面（ResetPasswordPage）
  - 注册时真实姓名和手机号改为可选字段
- **仪表盘页面改版**：DashboardPage 全面重写，UI 和信息展示优化
- **收件箱消息详情改版**：InboxMessageView 重新设计，信息展示更清晰
- **当事人详情页 UI 改版**：对齐合同详情页设计风格
- **侧边栏和导航优化**：菜单配置重构，新增工具子菜单，导航体验改进
- **列表页统一规范**：统一 ID 列展示，分页改为客户端分页

#### 新增组件

- **项目级共享组件**：CommandPalette、DataTable、EmptyState、InfoGrid、CollapsibleCard、DetailCardGrid、InlineCrudTable、PageFooter、RadioPillGroup、StatusBadge、Timeline、TopbarIcons
- **shadcn/ui 组件**：Breadcrumb、Calendar、Checkbox、Collapsible、Command、Popover、ScrollArea、Sheet、Switch、Textarea

#### 移除

- **公开营销页面**：删除首页（HomePage）、定价页（PricingPage）、教程页（TutorialPage）及所有相关组件（HeroSection、FeaturesSection、CTASection、CaseFlowDemo、ChatRecordDemo、SMSFlowDemo、TechStackSection、Footer、Navigation 等）
- **TopNavMenu 组件**：删除顶部导航菜单组件
- **AnimatedLink、RippleButton 组件**：清理未使用的旧组件
- **PublicLayout**：删除公开页面布局

#### 依赖变更

- 新增 `cmdk`：CommandPalette 命令面板
- 新增 `radix-ui`：统一 Radix UI 组件包
- 新增 `react-day-picker`：Calendar 日期选择器

### 后端

#### 新增

- 全局搜索 API（search_api）：支持跨模块业务数据检索
- 任务队列 API（task_queue_api）：支持查看和管理后台异步任务
- 消息来源 API（message_source_api）：MessageSource 完整 CRUD
- 法院短信 API 增强：支持查看、重试等操作
- 快递查询 API（express_query）：对接 EMS/顺丰查询
- 密码重置 API：支持忘记密码和重置密码流程
- 律师注册 API：完善注册流程
