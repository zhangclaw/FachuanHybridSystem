## [26.45.0] - 2026-05-07

### 前端

#### 新功能

- **批量文档分析**：全新 BatchAnalysisDialog 组件，支持拖拽文件夹上传 .doc/.docx 文件，自动提取案号、法官、书记员等信息，异步执行分析任务
- **批量分析进度卡片**：BatchProgressCard 实时展示分析进度，支持 SSE 推送
- **批量分析历史面板**：BatchHistoryPanel 查看历史批量分析记录
- **工作台命令面板**：WorkbenchCommandPalette 支持快捷键触发，快速执行常用操作
- **建议提示组件**：SuggestedPrompts 在对话区域展示常用提示词，降低使用门槛
- **消息气泡语法高亮**：代码块支持语法高亮渲染
- **消息导出功能**：支持将对话内容导出
- **上下文用量 hook**：useContextUsage 提取为独立 hook，复用性更强

#### 修复

- **移动端侧边栏适配**：移除多余的折叠按钮和 X 关闭按钮，修复移动端布局问题
- **批量分析对话框布局**：上传文件后不再挤压 UI，Textarea 焦点环边框均匀
- **Virtuoso 消息列表空白**：修复嵌套在 ScrollArea 内无法渲染的问题
- **消息超 50 条刷新丢失**：修复工作台消息超过 50 条时刷新页面只显示前 50 条
- **空消息状态居中**：修复工作台空消息状态居中位置错误
- **CSV 下载按钮消失**：修复刷新后下载按钮丢失的问题
- **案例元数据汇总渲染**：代码块中的【案例元数据汇总】正确渲染为结构化内容
- **MCP client 代理问题**：绕过系统代理直连后端，解决局域网环境下的连接问题
- **侧边栏层叠问题**：主内容区加 z-10 层叠上下文，避免面包屑/标题与侧边栏重叠
- **Court SMS 删除按钮**：移除法院短信列表页单条删除按钮，避免误操作
- **SiliconFlow 兜底模型**：移除不可用时的兜底模型列表，避免误导
- **提醒表单校验**：放宽关联校验规则

### 后端

#### 新功能

- **批量文档分析**：完整后端实现，支持批量上传文档 → 异步提取文本 → AI 分析 → CSV 汇总报告，包含并发限流、结构化输出、自动重试
- **批量分析任务系统**：BatchJob 模型 + Celery 异步任务，支持进度追踪、汇总文件生成
- **文档文本提取器**：doc_extractor 模块，支持 .doc/.docx 文本提取，.doc 通过 LibreOffice 自动转换
- **DOC 转 DOCX 引擎**：独立 doc_converter 应用，抽取为可复用的文档转换服务
- **WKInfo URL 案例检索范围**：支持通过 WKInfo URL 定义案例检索的数据库范围
- **WK插件迁移**：WK私有 API 适配器迁移到 plugins/ 目录统一管理
- **工作台 Agent 重构**：移除通用助手（general agent），保留分诊 + 3 个专业助手（案件管理、合同管理、法律检索）
- **WXMP 应用**：新增 wechat_mp 应用模块

#### 修复

- **批量分析案号提取**：从文件头部和文档表格中提取案号，从尾部段落提取法官书记员
- **汇总报告格式**：改为 CSV 格式，结论取完整分析结论，无关案例自动跳过
- **汇总报告日期**：修复使用错误 timezone.utc 的问题
- **研究报告日期**：使用实际日期而非 LLM 编造的日期
- **openai_compatible 后端**：配置了 base_url 时自动启用
- **MCP client 重试**：token 获取加重试，避免瞬时 502 导致工具调用失败
- **文件上传限制**：DATA_UPLOAD_MAX_NUMBER_FILES 提高到 5000，支持批量上传大量文件
- **doc_extractor 重构**：委托 doc_converter 引擎，移除内联的 LibreOffice 转换逻辑
- **FieldFile 序列化**：修复 BatchJobOut schema summary_file 字段序列化报错
- **批量分析对话持久化**：分析完成后将汇总报告插入对话而非独立面板
- **DocConverterTool admin 报错**：修复虚拟模型（managed=False）查询报错 relation does not exist
