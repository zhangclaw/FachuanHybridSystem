## [26.50.4] - 2026-05-25

### 后端

#### 新功能

- **法律科技频道**：新增 7 个 RSS 源（LawSites、Artificial Lawyer、LawNext、36氪AI、InfoQ、机器之心、量子位）+ Playwright 爬虫，支持中英文法律科技关键词自动过滤
- **热搜源扩充**：新增抖音、36氪、澎湃新闻热搜采集器
- **热点翻译**：新增 `/topics/translate` 端点，调用 LLM 批量翻译英文标题为中文

#### 优化

- **热点采集性能**：法律科技频道 7 个 RSS 源改为 `ThreadPoolExecutor` 并行采集，速度显著提升
- **except Exception → 具体异常类型**：修复 151 个裸 `except Exception`（ratchet 1551→1550），覆盖 JSON 解析、ORM 操作、网络请求等场景
- **API 层四层架构合规**：3 轮修复将 `Model.objects` 查询从 API 层迁移至 Service 层，涉及 client、archive、search、task_queue、message_hub、reminder、lpr、case_import、client_import 等模块

#### 修复

- **一张网立案插件门控**：`has_court_automation_plugin()` 改为检查环境变量 `ENABLE_COURT_FILING_PLUGIN=1`，普通用户无此变量时整个立案 tab 隐藏
- **预存测试失败**：修复 django_q 连接泄漏 + FinalizedMaterial 孤立列（新增迁移 `0029`）
- **resubmit_task 中间件问题**：改用 `async_task` 直接调用，避免 `submit_task` 中间件包装
- **热搜源清理**：删除有反爬机制的微博和知乎来源，只保留有公开 API 的来源

### 前端
