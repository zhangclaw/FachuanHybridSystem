## [26.44.1] - 2026-05-06

### 前端

#### 新功能

- **工作台体验全面升级**：借鉴 Claude Code 模式，三项核心改进
  - 工具调用内联展示：流式期间和历史消息中，工具调用统一内嵌在助手气泡内，可折叠展开查看输入/输出详情，告别独立 ToolMessage 卡片的割裂感
  - Agent 活动指示器：流式期间顶部实时显示当前活动状态（"xxx 正在思考..."、"正在执行 yyy..."、"切换到 zzz..."），消除用户等待焦虑
  - 用户消息编辑重发：hover 用户消息显示编辑按钮，修改内容后 Enter 触发重发，后端自动截断后续消息
- **工作台可折叠会话侧边栏**：会话列表侧边栏支持收起/展开，收起时联动收起管理后台侧边栏，展开时恢复之前状态
- **会话侧边栏搜索**：新增搜索框，支持按会话标题模糊过滤
- **会话标题可编辑**：点击标题旁的编辑图标即可重命名会话
- **模型选择器简化**：从 Button 样式改为极简文字样式，更紧凑
- **侧边栏收起浮层增强**：分组菜单在侧边栏收起时改为 fixed 定位的弹出浮层，带箭头、毛玻璃背景、动画效果，点击外部关闭
- **侧边栏收起品牌标识**：收起状态下显示 "FC" 缩写标识
- **上下文用量指示器**：顶部栏新增 ContextUsageBar，展示当前会话 token 用量
- **日历弹窗优化**：简化关闭按钮，新增提醒创建/编辑/删除功能
- **当事人列表标识**：名称列增加我方/对方标签，区分当事人身份
- **合同详情文档生成**：文档生成 Tab 对接后端 API，支持从模板生成合同文档
- **Dashboard 页面增强**：仪表盘页面功能完善与布局优化

#### 修复

- **文档与提醒标签布局**：补充协议列表改为紧凑布局
- **合同表单优化**：底部留白修复，当事人选择支持搜索
- **合同 API 请求体**：创建/更新请求体包裹 payload 字段，与后端 schema 对齐
- **合同详情标签名**：与后端 admin 保持一致
- **Header 高度对齐**：管理后台侧边栏品牌区高度从 h-14 调整为 h-12，与工作台标题栏分割线对齐

### 后端

#### 新增

- **Pydantic AI Agent 框架**：工作台 Agent 从手写 agent loop 迁移到 Pydantic AI，支持 `agent.iter()` 流式迭代、MCP 工具自动发现、结构化工具调用
- **联网搜索工具**：注册 `web_search` MCP 工具（搜狗搜索引擎），支持 Agent 联网搜索最新信息、法规、新闻等，国内直接访问无需 API Key
- **消息截断 API**：新增 `DELETE /sessions/{id}/messages/from/{msg_id}`，支持从指定消息位置截断后续消息，用于用户消息编辑重发

#### 修复

- **MCP Server 启动**：使用 `sys.executable` 替代硬编码 `python` 命令，避免虚拟环境路径问题
- **Ollama 模型列表**：不再自动发现，只返回 SystemConfig 中配置的模型
- **Agent 系统提示词优化**：强化工具调用意识，明确要求"必须使用工具，不要凭记忆回答"，特别强调联网搜索必须主动调用
- **实时日期注入**：系统提示词注入当前日期时间，避免模型回答过期日期
- **Agent 活动事件**：`meta` 事件增加 `agent` 字段，新增 `activity` 事件，修复 `agent_display_name` 未定义导致事件报错
