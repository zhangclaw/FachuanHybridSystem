### 关键技术决策

- **跳过 multipart 文件上传**：律师执照 PDF、头像等文件上传端点不纳入 MCP（MCP 场景不实用）
- **跳过认证端点**：login/logout/register 等不纳入（MCP Client 已内置 JWT）
- **命名冲突解决**：`generate_complaint`（core vs documents）→ core 改名 `generate_poi_complaint`；`health_check`（doc_converter vs automation）→ doc_converter 改名 `doc_converter_health_check`

### 工具注册架构

```
tool function → sub-package __init__.py → tools/__init__.py → server.py mcp.tool()
```

`server.py` 与 `tools/__init__.py` 均包含 376 个 import + `__all__` 条目，对齐验证通过。
