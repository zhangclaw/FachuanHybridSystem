# v26.52.6 (2026-06-14)

## 要素式转换核心功能私有化

出于法律合规风险考量，将肇庆中院 znszj 要素式转换核心客户端迁移至私有 plugins 子模块（FachuanPlugins），开源版本保留模块入口但功能降级：点击 admin 入口时自动跳转至肇庆中院官网。

---

## 一、架构变更

| 改动 | 说明 |
|------|------|
| `ZnszjClient` 迁移至 `plugins/doc_convert/` | 私有子模块，开源仓库零 znszj 实现引用 |
| `DocConvertService` 迁移至 `plugins/doc_convert/` | 随客户端一起迁移 |
| `has_doc_convert_plugin()` 插件检测 | 子模块 `plugins/__init__.py` 中新增 |
| `znszj_loader.py` 重构 | 改为从插件动态加载客户端 |

---

## 二、Admin 降级行为

| 场景 | 行为 |
|------|------|
| 插件已安装 | 正常渲染要素式转换工作台 |
| 插件未安装 | `has_view_permission` 返回 False，Django 自动重定向到 app index |
| 侧边栏入口 | 始终保留可见 |

---

## 三、API 降级行为

| 端点 | 无插件时行为 |

| `POST /convert` | 503 错误（已有降级路径，无需新增检测） |
