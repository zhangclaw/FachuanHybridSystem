# v26.53.5 (2026-06-22)

## 修复

### 暗黑模式全面修复

本分支彻底修复了 Django 6 Admin 后台的暗黑模式，将 250+ 处硬编码颜色替换为 CSS 变量，并补充了表单元素的暗黑模式覆盖。

**核心改动：**

- **恢复暗黑模式开关**：在 `base_site.html` 中恢复 Django 6 的暗黑模式切换功能（`06f13fb`）
- **全局硬编码颜色替换**：73 个文件、250+ 处硬编码颜色（`#fff`、`#333`、`rgb()` 等）替换为 `var(--fc-*)` CSS 变量（`55844ce`）
- **表单元素覆盖**：在 `fachuan_dark_mode_vars.css` 中新增对 `input`、`textarea`、`select`、`.submit-row`、`.module`、`fieldset`、`.collapse summary` 的暗黑模式样式，并使用 `!important` 确保优先级（`304388e`、`0b12aff`）
- **fieldset/collapse 修复**：修正 fieldset 边框颜色和 collapse summary 背景色（`31ea597`、`816b226`）

**涉及模块（按应用分组）：**

| 应用 | 修复文件数 | 主要内容 |

| cases | 11 | 案件详情页（按钮、卡片、标签页、时间线、当事人、文档等） |

| automation | 8 | 文档处理器、爬虫任务、法院短信、快速下载等 Admin 页面 |
| core | 5 | 系统配置、案件处理中心、Redis 队列等 |
| finance | 4 | LPR 计算器表单和脚本 |
| 其他 | 12 | 客户、聊天记录、合同审查、云存储、证据、PDF 拆分、诉讼AI等 |
