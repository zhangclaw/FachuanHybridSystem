## 技术模式

| 改造类型 | 方法 |
|---------|------|
| sync ORM → async ORM | `.objects.filter().first()` → `await .objects.afirst()` |
| sync 视图 → async 视图 | `def view()` → `async def view()` |
| sync_to_async → native async | `sync_to_async(service.method)()` → `await service.amethod()` |
| AsyncClient 泄漏 | 添加 `finally: await client.close()` |
| asyncio.run() 嵌套 | `asyncio.run(chain.arun())` → `await chain.arun()` |
| 串行 → 并发 | 循环调用 → `asyncio.gather(*tasks)` |
| sync 文件 I/O | `open()` → `asyncio.to_thread(open, ...)` |
