### 改造模式

```python
# 改造前
def my_view(request, ...):
    result = service.some_method(args)
    return result

# 改造后
async def my_view(request, ...):
    result = await sync_to_async(service.some_method, thread_sensitive=False)(args)
    return result
```

- 文件 I/O：`await asyncio.to_thread(lambda: ...)`
- ZIP 操作：`await asyncio.to_thread(lambda: zipfile.ZipFile(...).extractall(...))`
- 已有 async 函数保持不变
