# v26.51.6 - 2026-06-05

## 后端

### 新功能

- **Apache POI 文档生成微服务**：在 `java-services/poi-service/` 引入 Java Spring Boot + Apache POI 微服务，通过 HTTP 与 Django 集成，提供高质量 DOCX 文档生成能力
  - 起诉状生成（合并单元格表格、样式段落、落款对齐）
  - 尽调报告生成（样式表格、页眉页脚、分节符、水印）
  - 归档文书生成（案卷封面、结案归档登记表、卷内目录）
  - 通用 .docx 模板渲染
  - Django 集成：POIServiceClient HTTP 客户端 + Ninja API 端点（`/api/v1/poi/`）

### 优化

- **POI Service 全面整改**（技术栈升级 + 架构优化）
  - Java 21 → 25 (LTS)，Spring Boot 3.4.5 → 3.5.14，Apache POI 5.3.0 → 5.5.1
  - 目录结构调整：`poi-service/` → `java-services/poi-service/`
  - Dockerfile 重写：Alpine → Ubuntu Noble（中文字体）、非 root 用户、HEALTHCHECK、JVM 调优
  - Controller 拆分：DocumentController → DocumentController + ArchiveController + TemplateController
  - Model 类启用 Lombok @Data（删除 ~200 行手写 getter/setter）
  - 未使用 import 清理

- **Maven 多模块改造**：引入父 POM 统一版本管理，新建 `shared/common-models`（共享 DTO）和 `shared/poi-utils`（共享 POI 工具），DocxHelper 迁移至共享模块

- **测试文件整理**：Python 集成测试统一移至 `java-services/tests/`，测试输出目录通过 .gitignore 排除

### CI/CD

- **新增 Java 服务 CI**（`java-services-ci.yml`）：JDK 25 编译 + Maven 打包验证，仅 `java-services/**` 路径变更时触发
